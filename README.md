## 📌 Descripción

El **Sistema de Evaluación Docente** es una aplicación diseñada para recolectar, analizar y gestionar evaluaciones realizadas a profesores dentro de una institución educativa.

Permite mejorar la calidad educativa mediante la retroalimentación continua.

---

## 🚀 Características

- 📝 Encuestas dinámicas de evaluación  
- 👨‍🏫 Evaluación de docentes por estudiantes  
- 📊 Resultados y estadísticas  
- 🔐 Control de acceso por roles (admin, alumno)  
- 📅 Periodos de evaluación configurables  
- 📈 Reportes por docente  

---

## 🛠 Tecnologías utilizadas

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP / Node.js / Java (según implementación)  
- **Base de datos:** MySQL
